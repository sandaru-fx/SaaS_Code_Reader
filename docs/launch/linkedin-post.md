I spent 10 days building CodeRider — and it's live today.

The problem: understanding unfamiliar code is still too slow. You read file after file and try to reconstruct the architecture in your head.

The solution: a visual code workspace that turns any file or snippet into an AI-generated flowchart + plain-English explanation.

What you can do:
→ Open local folders in the browser (files stay on your machine)
→ Paste a snippet for instant analysis
→ Get Mermaid architecture diagrams + step-by-step logic breakdowns

Tech: Next.js, Gemini 1.5 Flash, Tailwind, Vercel — built as a $0 MVP.

Try it: https://coderider.vercel.app
GitHub: https://github.com/sandaru-fx/SaaS_Code_Reader

If you're a developer who's ever been lost in someone else's codebase — I'd love your feedback.
