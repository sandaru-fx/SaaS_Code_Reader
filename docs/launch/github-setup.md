# GitHub repo polish — manual checklist

No API required. Do this in the GitHub web UI.

## 1. Repository description

Go to: https://github.com/sandaru-fx/SaaS_Code_Reader

Click the **gear icon** next to "About" → set:

**Description:**
```
AI-powered visual code workspace — flowcharts & explanations from local folders or pasted snippets. Next.js + Gemini.
```

**Website:**
```
https://coderider.vercel.app
```

## 2. Topics (tags)

In the same About section, add topics:

```
nextjs
typescript
gemini
mermaid
ai
saas
developer-tools
code-visualization
tailwindcss
vercel
```

## 3. Pin the repository

On your GitHub profile → **Customize your pins** → pin `SaaS_Code_Reader`.

## 4. Social preview

GitHub auto-generates from README. The `og-image.png` in README helps when the link is shared.

## 5. Optional — enable Discussions

Repo **Settings** → **General** → **Features** → enable **Discussions** for user feedback.

## 6. Optional — Issues template

Keep Issues enabled for bug reports and feature requests from launch traffic.
