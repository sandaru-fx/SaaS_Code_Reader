## Description (short)

CodeRider turns files and snippets into visual architecture flowcharts and step-by-step AI explanations. Open local folders or paste code — no repo upload required.

## Description (long)

Understanding unfamiliar code is slow. CodeRider gives you a browser workspace where you open a local project or paste a snippet, click Analyze, and get a Mermaid flowchart plus a plain-English walkthrough.

**Features:**
- Local folder mode (File System API)
- Quick Paste for snippets
- Syntax-highlighted code viewer
- AI-powered diagrams via Gemini
- Free to try

Built in 10 days as a solo MVP on Next.js + Vercel.
